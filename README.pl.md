## Użycie

Stwórz albo edyuj plik `src/config/config.js`.
```
module.exports = {
  token: "___WKLEJ_TUTAJ_SWOJ_TOKEN___",
  prefix: "!",
}
```

token — twój token bota (powinnien zawsze być prywatny)

prefix — prefix dla kommend, na które reaguje bot
